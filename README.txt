You know what? With this plugin installed, the Bengali text on your website will be as clear as the text in a book.The texts of the website will become clear and beautiful. As a result, the content of the website will be neat and clean.

== Description ==

Bangla language people are regularly facing Bengali font problem in WordPress. Especially Bangla sites are failing to make their fonts look beautiful. However, this plugin is not the first of ours, but other plugins have also had some issues. So we further developed the plugin and brought it to the market completely free. Now you just activate the plugin and enjoy!

== Installation ==

1. Unpack the entire contents of this plugin zip file into your wp-content/plugins/ folder locally
2. Upload to your site
3. Navigate to wp-admin/plugins.php on your site (your WP Admin plugin page)
4. Activate this plugin

OR you can just install it with WordPress by going to Plugins > Add New > and type this plugin’s name
